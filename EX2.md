## Ex.Name: Print Sorted Elements of an Array
## Date: 10/11/25
## Aim:
To write a C++ program to print the sorted elements of an array.

## Algorithm:
Start the program.

Input the elements of the array.

Display the array before sorting.

Apply a sorting technique (for example, sort() from STL or a custom algorithm like Bubble Sort/Quick Sort).

Display the array after sorting in ascending order.

Stop the program.

## Program:
```
void print(int arr[]){
    cout<<"After Sorting the Array:"<<endl;
    for(int i=0;i<5;i++){
        cout<<arr[i]<<" ";
    }
}
```
